Just applying the cake metaphor to the programming domain:


          candles      !   !   !   !   !      IDE? bugs?
 decorative icing   ~~~~~~~~~~~~~~~~~~~~~~~   source code
   top cake layer   |||||||||||||||||||||||   compiled source
     filler icing   -----------------------   runtime
